# Math Operations

## Description
This repository contains Python functions for basic math operations (addition and subtraction) and corresponding test files.

## Folder Structure
- **src**: Contains the production code (`math_operations.py`) for math operations.
- **tests**: Contains pytest files (`test_add.py` and `test_subtract.py`) for testing the math operations.

## Usage
1. Clone the repository:
    ```bash
    git clone https://github.com/lakshmi1212/Math.git
    ```
2. Navigate to the repository:
    ```bash
    cd Math
    ```
3. Install dependencies (if required):
    ```bash
    pip install pytest
    ```
4. Run tests:
    ```bash
    pytest tests/
    ```

## Workflow
1. Add production code to the `src` folder.
2. Add test files to the `tests` folder.
3. Ensure all tests pass before committing changes.
